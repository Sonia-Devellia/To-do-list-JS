# To-Do List JavaScript

Ce projet est une **To-Do List** simple réalisée pour pratiquer et appliquer les concepts appris en **JavaScript**.  

Le **HTML** utilisé dans ce projet provient d’un exercice de [Grafikart](https://www.grafikart.fr/) et **n’a pas été écrit par moi**.  

Le **CSS** est minimal et l’objectif **n’était pas de créer un design joli**, mais seulement de rendre l’interface fonctionnelle pour tester le JavaScript.  

## Fonctionnalités

- Ajouter une tâche à la liste
- Marquer une tâche comme terminée
- Supprimer une tâche
- Persistance locale (optionnelle selon l’implémentation)

## Technologies utilisées

- **JavaScript** : pour la logique et l’interaction avec l’utilisateur  
- **HTML** : structure de la page issue d’un exercice de Grafikart  
- **CSS** : stylisation minimale, juste pour rendre le projet fonctionnel
